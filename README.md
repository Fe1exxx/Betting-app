# Betting App Landing Page

Адаптивный одностраничный сайт. Полностью на HTML, CSS и vanilla JavaScript.

## Особенности:
- Полностью адаптивная верстка (mobile-first)
- Интерактивное мобильное меню с overlay
- Swiper.js для каруселей (матчи, отзывы)
- FAQ с раскрывающимися блоками
- Кросс-браузерная совместимость
- Оптимизированная производительность

## Технологии:
- HTML5
- CSS3 (Flexbox, медиа-запросы)
- JavaScript (ES6+)
- Swiper.js
- Google Fonts (Inter, Poppins)
